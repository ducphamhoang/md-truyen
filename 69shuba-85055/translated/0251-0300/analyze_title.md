## Các thay đổi đã thực hiện

### 1. Format thống nhất
- **Trước:** Hỗn loạn giữa `<h1>Chương xxx</h1>` + `<p># Tiêu đề: ...</p>` và `<h1>Chương xxx: yyy</h1>`
- **Sau:** Tất cả đều là `<h1>Chương xxx: yyy</h1>`

### 2. Loại bỏ nội dung dư thừa
Đã xóa các dòng không cần thiết:
```
# Tiêu đề: Chương xxx: ...
# URL: https://www.69shuba.com/...
# Truyện: ...
# Sách: ...
============================================================
```

### 3. Sửa số chương sai
- **Lỗi phổ biến:** Số chương trong tiêu đề thấp hơn 2 đơn vị so với tên file
- **Ví dụ:** 0251.html có tiêu đề "Chương 249" → Sửa thành "Chương 251"

### 4. Thêm tiêu đề cho chương thiếu
Các chương 0276, 0281, 0299 trước đó không có thẻ `<h1>` đã được bổ sung.
